QuestOfRealms Overview

The app is split into two parts – game designer and game client. They share core technologies such as the sails application framework and jQuery / jQueryUI for the UI, a Node.js server with MongoDB for the database.

Game Designer

This is a multi-user app for designing games. A game template is created, from which playable game instances can be cloned.

See Game Designer Overview for what it looks like and how to use it.

See Game Designer Development Guide for how it works and how to update it.
