QuestOfRealms setup procedure, Ubuntu 14

Install and configure git

From the howtoforge github install procedure. 
sudo apt-get install git
git config --global user.name "user_name"
git config --global user.email "email_id"
You can use the commandline tools but if you prefer a GUI you can check out a few. 
sudo apt-get install gitk
sudo apt-get install git-gui
sudo apt-get install giggle

Download the QuestOfRealms project

Using this procedure. 
git clone https://github.com/SimonLogan/QuestOfRealms
Open the project using giggle 
cd QuestOfRealms
giggle

Install nodejs

Using this procedure.
On Debian-based systems like ubuntu, node needs to be installed as nodejs 
sudo apt-get update
sudo apt-get install nodejs

Install the node dependencies of QuestOfRealms

Install npm

Installing npm with "sudo apt-get install npm" seemed to cause version conflicts when I came to install sails.
Installing a version of npm that is compatible with sails worked: follow this procedure.

Install sails

Follow this procedure. QuestOfRelams/package.json lists the dependencies the app requires. Install these with npm 
cd QuestOfRealms
npm install

install the waterline mongodb adapter

sudo npm install sails-mongo@0.9.x

Install mongodb

Follow this procedure.

Using QuestOfRealms

Start mongodb 
sudo service mongod started
Start the app 
cd QuestOfRealms
sails lift
You should see
