QuestOfRealms setup procedure, Windows 10

Install and configure git

The easiest way is the GitHub Desktop.

Download the QuestOfRealms project

The easiest way is to choose the "Clone" option in GitHub Desktop Open the project using giggle 
cd QuestOfRealms
giggle

Install nodejs

Using this procedure.
On Debian-based systems like ubuntu, node needs to be installed as nodejs 
sudo apt-get update
sudo apt-get install nodejs

Install the node dependencies of QuestOfRealms

Install npm

Installing npm with "sudo apt-get install npm" seemed to cause version conflicts when I came to install sails.
Installing a version of npm that is compatible with sails worked: follow this procedure.

Install sails

Follow this procedure. QuestOfRelams/package.json lists the dependencies the app requires. Install these with npm 
cd QuestOfRealms
npm install

install the waterline mongodb adapter

sudo npm install sails-mongo@0.9.x

Install mongodb

Follow this procedure.

Using QuestOfRealms

Start mongodb 
sudo service mongod started
Start the app 
cd QuestOfRealms
sails lift
